ClitorizWeb API Documentation


WARNING: MISUSE AND ABUSE OF THE API WILL RESULT IN A TIME LIMIT, A BAN OR OTHER RESTRICTIONS.


Users API


http://clitoriz.cf/api/users/getinfo.php?user=goom

Gets information for a specified user. (CSS data is encoded using base64 due to JSON limitations)


http://clitoriz.cf/api/users/list.php

Gets the list of all CW users.

 

Forums API


http://clitoriz.cf/api/forums/getsubforums.php

Gets the current subforums


http://clitoriz.cf/api/forums/getthreads.php?id=1

Gets all of the threads in a sub forum


http://clitoriz.cf/api/forums/thread.php?id=1

Gets the replies and the information about a thread
